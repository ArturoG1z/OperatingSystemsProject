Remember that the program works with two configuration files, one for configuring memory called 'memory.conf' and the other for commands called 'commands.conf'

If the number of virtual pages is not entered, 64 is assumed

If the number of page frames is not entered, 32 is assumed

If the page size is not entered, 4096 is assumed

If for some reason time is put in memory (inMemTime) or time of last reference (lastTouchTime) in virtual pages without mapping these values will be set to -1

To reload data from the configuration files press the exit button in the simulation and load from this load button
